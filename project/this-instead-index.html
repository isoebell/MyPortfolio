This Instead Index

An information resource, web design and web development

This Instead Index is never finished. This instead is a response to the corners of the internet that produce, circulate and encourage discrimination. This instead challenges online spaces where bigotry thrives. This instead provides an information container of resources and links that are the antithesis of bigotry and discrimination. This instead is a space to learn more about topics, destress after a doom scroll or expand your perspective.

Suggested resources are welcomed here.

Currently, this site exists as a proof on concept. It is live and functioning, but has its kinks. I'm excited by the capacity that this project has to transform and grow.
